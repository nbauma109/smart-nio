# smart-nio
An extension of Java NIO API for filesystems based on compressed archives (tar.gz, tar.bz2, tar.xz, 7z,...)

## Why this project

With regular Java NIO API, it is only possible to exploit zip-compressed archives as filesystems, such as jar, war, ear, etc...
This project aims at providing support for non-compressed archives (tar) and other types of compressed archives such as tar.gz, tar.bz2, tar.xz, 7z,...
